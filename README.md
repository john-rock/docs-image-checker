## Running Locally

To set up and run the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/john-rock/docs-image-checker.git
   cd docs-image-checker
    ```
2. Install dependencies
   ```bash
   npm install
    ```
3. Run script
   ```bash
   node index.js
   ```
